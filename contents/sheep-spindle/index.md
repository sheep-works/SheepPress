# SheepShuttle

SheepWeaver でできた JSON をさらに活用するためのツールです。

コンセプトは **杼（Shuttle）**。

データの分割や結合、TM ファイル化など、「制作用データ」を「管理用データ」として活用しやすくします。

## 主な機能

- **分割・結合**: JSON の分割・結合により分業をサポート
- **TM/TB 変換**: JSON を TM/TB ファイルに変換して、他のツールでも活用できるように
- **JSONL 変換**: SheepLint をはじめとする、AI ツールで使いやすい JSONL 形式への変換もサポート
