# SheepWeave

VS Code 拡張機能で実現する翻訳支援ツール（CAT ツール）です。

コンセプトは **機織り（Weave）**。

翻訳の糸を紡ぎ、美しい布を織り上げます。

## 主な機能

- **翻訳メモリ・用語集の活用**: 翻訳メモリとの類似や含まれる用語など、CAT ツールとしての基本機能をサポート
- **強力な置換＆補完**: プレーンテキストベースなので置換が高速。さらに AI による補完とも高相性
- **ローカル処理**: 解析から翻訳までローカルで完結。VS Code 環境さえあれば、どこでも軽量に実行可能
